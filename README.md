# Anti-span System

Невеличкий мікросервіс, створений з метою запобігання спаму, використання ефективних антиспам-фільтрів та налаштуванні надійного захисту.

# Технології, які використовуються

## Встановлення залежностей
```properties
$ yarn install
```

## Компіляція та запуск проєкту
```bproperties
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Запуск тестів

```properties
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
